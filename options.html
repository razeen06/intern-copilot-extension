Application Tracker Settings

API Token

The easiest way to connect is the "Connect Extension" button on your Application Tracker dashboard (sign in to the web app, then click it on the dashboard page) -- it sends your token here automatically, no copy-pasting needed.

Prefer to do it by hand, or the button isn't working? Paste your token below manually instead (find it by visiting /api/token while signed in). Applications are always saved locally to this extension too, so tracking still works offline either way.

Sites you've enabled detection on

Beyond the built-in job boards, you can enable detection on any other career page from the extension's popup when it's not covered yet. Sites you've approved that way are listed here and can be removed anytime.
